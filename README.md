# Matlab-DDE-TracePro
由于网络资源较少，贴主学习Matlab通过DDE控制TracePro走了很多弯路，但最后实现了基础的控制功能。贴主已经不做光学设计了，基于互联网分享精神，现此开源相关代码，希望帮助后来者入门TracePro自动化减少痛苦。

我的学习渠道：1.论坛：吾爱光设、光行天下等，检索DDE；2.知网中文论文，检索DDE、TracePro；3.TracePro官网、Git上其它贴主的分享；4.淘宝等其它平台售卖的电子资料。通过AI学习、生成基础代码。

基本控制办法：同时打开Matlab和TracePro界面，然后运行Matlab代码即可。DDE通过4个基本语句与TracePro连接，实现TracePro具体功能要查询软件内附带的宏命令手册。

上传的代码包括：建立连接、自动化建模、光学追迹、光学分析、后处理等部分。
